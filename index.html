1.1. Variabel dalam Javascript

Variabel adalah keyword yang digunakan untuk menyimpan suatu data. Variabel dapat dideklarasi dengan 3 cara, yaitu:

  • var = Variabel 1995 s/d 2015 yang sering digunakan. Setelah itu, variabel yang sering digunakan adalah const dan let
  • const = Variabel yang menyimpan data tetap, tidak berubah-ubah.
  • let = Variabel yang datanya dapat berubah

1.2. Variabel Let

variabel let dikenalkan pada tahun 2015. Data yang dideklarasi dengan variabel ini tidak dapat dideklarasi kembali.

Data yang ingin digunakan, harus dideklarasikan dengan let terlebih dahulu.

Variabel let memiliki Block Scope (Akan dijelaskan di slide selanjutnya)

1.3. Variabel Const

Data yang dideklarasi dengan variabel const tidak dapat diubah-ubah.

Variabel const harus langsung diberikan value.

Gunakan variabel ini ketika ingin membuat Array, Object, Function, dan RegExp.

2.1. Javascript Operators

Javascript mengenal operator matematika seperti kabataku (Kali Bagi Tambah Kurang).

Ada beberapa tipe Javascript Operators, seperti Arithmetic Operators, Assignment Operators, Comparrison Operators, String Operators, Logical Operators, Bitwise Operators, Ternary Operators, dan Type Operators.

2.2. Arithmetic Operators

Di bawah ini adalah beberapa Arithmetic Operators:

  • + = Penambahan. Contoh (2+2)
  • - = Pengurangan. Contoh (3-2)
  • * = Pengalian. Contoh (3*2)
  • ** = Pemangkatan. Contoh (5**2)
  • / = Pembagian. Contoh (10/2)
  • ++ = Penambahan 1 pada variabel. Contoh ( let i = 0; i++)
  • -- = Pengurangan 1 pada variabel. Contoh (let i = 10; i-- )

3.1. Tipe Data dalam Javascript

Javascript Memiliki 8 jenis data, yaitu

  • String
  • Number
  • Bigint
  • Boolean
  • Undefined
  • Null
  • Symbol
  • Object

Object juga memiliki 3 tipe, yaitu:

  • Object
  • Array
  • Date

3.2. Tipe Data Dinamis

Javascript memiliki tipe data yang dinamis, artinya suatu data dapat berubah-ubah tergantung kapan ia dideklarasikan kembali, contoh:


  • let x; // Sekarang, x telah dideklarasi dengan nilai awal Undefined
  • x = 5; // Nilainya berubah dari Undefined menjadi 5
  • x = 'John'; // Sekarang, nilai X berubah lagi dari 5 menjadi 'John'

Nilai tersebut dapat berubah-ubah dikarenakan keyword yang digunakan untuk deklarasi variabel adalah let, bukan const.

3.3. Tipe Data (Strings)

Strings adalah tipe data berupa kata-kata yang dituliskan.

Ada dua cara untuk menentukan tipe suatu data menjadi strings, yaitu dengan menggunakan tanda petik dua ("") dan tanda petik satu (''), contoh:


  • let carName1: "Muhamad Aqil Maulana" // menggunakan tanda petik dua
  • let carName2: 'Muhamad Aqil Maulana' // menggunakan tanda petik satu
